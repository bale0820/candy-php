[지금까지 설치한 것]

백엔드: E:/projects/candy-php

1. Laravel Sanctum
설치 명령:
composer require laravel/sanctum

설치 목적:
- Vue SPA와 Laravel API를 세션 쿠키 기반으로 로그인 처리하기 위해 설치
- /sanctum/csrf-cookie 사용
- auth:sanctum 미들웨어 사용
- API에서도 세션 인증을 사용할 수 있게 함

설치 후 생성/게시된 파일:
config/sanctum.php
database/migrations/2026_06_21_065148_create_personal_access_tokens_table.php

실행한 게시 명령:
php artisan vendor:publish --provider='Laravel\Sanctum\SanctumServiceProvider'

실행한 마이그레이션:
php artisan migrate


2. Composer 패키지 변경
composer.json에 추가됨:
laravel/sanctum

composer.lock에 추가됨:
laravel/sanctum v4.3.2


3. 프론트: E:/projects/vue/php-frontend

새로 npm install 한 패키지는 없음.

기존 사용 중인 주요 패키지:
vue
vite
typescript
@vitejs/plugin-vue
vue-tsc

즉 프론트 쪽은 패키지 추가 설치 없이 기존 Vue/Vite 구성에서 코드만 수정함.


4. 카카오/네이버 로그인 관련

추가 설치한 패키지는 없음.

Socialite 같은 패키지는 설치하지 않았음.
직접 Laravel Http 클라이언트로 OAuth 처리함.

사용한 Laravel 기본 기능:
Illuminate\Support\Facades\Http
Illuminate\Support\Facades\Auth
Illuminate\Support\Facades\Hash

구현 방식:
- 카카오 authorize URL 직접 생성
- 네이버 authorize URL 직접 생성
- callback에서 code를 access_token으로 직접 교환
- access_token으로 프로필 직접 조회
- Laravel 세션 로그인 처리
